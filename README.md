Contexto

Este projeto foi desenvolvido utilizando conceitos de Aprendizagem Ativa com Inteligência Artificial e NotebookLM.

O objetivo foi criar uma base de conhecimento estruturada sobre os setups operacionais que mais se alinham ao meu estilo de trading, consolidando informações obtidas por meio de vídeos, artigos acadêmicos, estudos de mercado e materiais especializados.

Os setups estudados foram:

* Larry Williams 9.1
* Larry Williams 9.2
* Larry Williams 9.3
* Larry Williams 9.4
* Setup 123
* Turtle Trading
* Dave Landry


Objetivos de Estudo

* Entender a lógica operacional de cada setup.
* Identificar contextos ideais de utilização.
* Comparar vantagens e limitações.
* Construir uma base de consulta rápida para futuras operações.
* Criar prompts reutilizáveis para acelerar revisões e aprofundamentos.
